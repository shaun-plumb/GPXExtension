WHAT IS THIS?
═════════════════════════════════════════════════════════════════════════════

A Chrome extension that exports Google Maps routing results as GPX files
(GPX is a standard format for GPS/navigation devices).

Converted from Java (Main.java) to a fully functional Chrome extension.


QUICK START (5 MINUTES)
═════════════════════════════════════════════════════════════════════════════

1. BUILD
   $ cd ~/GPXExtension
   $ ./build.sh

2. INSTALL IN CHROME
   • Go to: chrome://extensions/
   • Turn ON "Developer mode" (top-right toggle)
   • Click "Load unpacked"
   • Select folder: ~/GPXExtension/build
   • Click "Select"

3. TEST
   • Go to: https://www.google.com/maps/dir/
   • Create a route (e.g., Amsterdam → Rotterdam)
   • Click the extension icon in the toolbar
   • Click "Export to GPX" button
   • A GPX file downloads automatically!

That's it! The extension is now installed and working.

HOW IT WORKS
═════════════════════════════════════════════════════════════════════════════

1. You create a route on Google Maps
2. Click the extension icon
3. Click "Export to GPX"
4. The extension:
   • Reads the route data from the Google Maps URL
   • Parses the encoded waypoint coordinates
   • Generates a standard GPX file
   • Downloads it to your computer
5. You can use the GPX file with:
   • GPS devices (Garmin, etc.)
   • Navigation software
   • GPS viewers (Google Earth, etc.)
   • Any software that supports GPX format

All processing happens locally in your browser. No data is sent anywhere.

TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Extension won't load?
  → Make sure "Developer mode" is ON in chrome://extensions/
  → Run ./build.sh first
  → Select ~/GPXExtension/build folder (not root folder)

Export button doesn't work?
  → Make sure you're on a Google Maps routing page (URL has /dir/)
  → Create a complete route with start and end points
  → Wait for Google Maps to fully load

No file downloads?
  → Check your Downloads folder
  → Open browser console (F12) and check for errors
  → Try a simpler route

For more help, see INSTALLATION.md
